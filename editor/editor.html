        About

        WMap Editor v1.0

        GUI tool to edit/transcode WeatherMap configs.

        Author Vitaly Chekryzhev (13hakta), 2017.

        Feel free to donate: PayPal, Yandex.

        Supports / features

        • Easy to drag&drop nodes movement
        • Templated nodes and links
        • Background color and image
        • Convert from and to WeatherMap format
        • Load and save to native WeatherMap configs
        • Editor for nodes, links, scales, fonts, templates, map properties
        • Links with VIA, multiple VIA and curved/angled styles
        • Gently handles unsupported tags (excepting templated objects)
        • All types of time objects: min/max/generated time
        • Display title
        • Grid contrasting to background
        • Snap to grid
        • Scales are drawn in classic style
        • Optionally can decrease redraw intense on movement
        • SET hidezero and hidepercent for scales

        Unsupports

        • Links are displayed as lines without bandwidth labels
        • Text only in predefined mono fonts
        • Scale styles (only classic), tagged
        • Node tags: LABELANGLE SET USEICONSCALE ZORDER
        • Link tags: BWLABELPOS BWSTYLE COMMENTFONT COMMENTPOS COMMENTSTYLE INBWFORMAT INCOMMENT ININFOURL INNOTES INOVERLIBCAPTION INOVERLIBGRAPH OUTBWFORMAT OUTCOMMENT OUTINFOURL OUTNOTES OUTOVERLIBCAPTION OUTOVERLIBGRAPH SET SPLITPOS ZORDER
        • Global tags: IMAGEURI INCLUDE KEYSTYLE KILO SET
        • IDs are not checked for interference
        • Comments are not preserved on converting
        • No undo, sorry. ;)

        Uses

        • JQuery v3
        • Bootstrap v4.1.1
        • Popper
